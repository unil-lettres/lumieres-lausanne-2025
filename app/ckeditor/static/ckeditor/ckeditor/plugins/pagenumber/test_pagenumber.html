CKEditor Page Number Plugin Test

This page tests the pagenumber plugin. Click the "Page Number" button in the toolbar to insert a page tag.

Test with English (default)

Test with French

Instructions for Testing

  1. Position your cursor in either editor
  2. Click the "Page Number" (or "Page numéro" in French) button in the toolbar
  3. Enter a page number in the dialog (e.g., "5")
  4. Click OK
  5. Verify that the tag <<5>> appears at the cursor position
  6. Test validation by:
    • Leaving the field empty and clicking OK (should show error)
    • Entering non-numeric text (should show error)

Expected Results

  • The button should appear in the toolbar with a page icon
  • Clicking it should open a dialog requesting a page number
  • Valid numeric input should insert <<number>> at cursor position
  • Invalid input should show validation errors
  • French editor should show French labels and messages
